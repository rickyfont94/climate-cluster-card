# Security Policy

## Scope

The Climate Cluster Card is a client-side Lovelace dashboard card. It runs entirely in the browser, has no backend, and stores no secrets or credentials. It only reads the Home Assistant state already available to the logged-in dashboard user.

## Supported versions

Only the latest released version is supported. Please update to the latest release before reporting an issue.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

Please report security issues privately through the repository's Security tab using "Report a vulnerability" (GitHub Private vulnerability reporting). Do not open a public issue for security reports.

You will get a response as soon as possible. Once a fix is available, it will be included in the next release.
